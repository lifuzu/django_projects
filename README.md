### Start coverage
```
$ pipenv run coverage
$ pipenv run cov_report
```

### Migrate database
```
$ python manage.py makemigration
$ python manage.py migrate
```

### Load fixtures data
```
$ python manage.py loaddata leads
```

### Run the frontend and server together
```
$ npm run dev
$ python manage.py runserver
```
